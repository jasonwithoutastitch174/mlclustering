## 🛠️ How to use the app

The interface makes complex data science tasks accessible. Follow this guide to process your first set of data:

### 1. Upload your file
The app accepts data in the CSV format. Look for the file upload button on the left side of the screen. Select your file from your computer. The app will show a preview of your data once it loads.

### 2. Choose your method
You can pick between different grouping techniques. K-Means works well for simple groups, while methods like DBSCAN help if your data has unusual shapes. Use the menu to toggle between these options.

### 3. Simplify your data
If your file contains too many columns, use the dimensionality reduction section. Tools like PCA help squish your data into a visual format. This makes it easier to see clusters on a flat screen.

### 4. Adjust settings
Each method has specific sliders. Move these sliders to change how the app calculates groups. You will see the visual chart update as you move these controls.

### 5. Check the metrics
The app calculates validation scores. These numbers tell you if your groups make sense. High scores indicate that your data points fit well within their assigned groups.

### 6. Save your results
Once you feel happy with the configuration, click the export button. The app creates a new file. This file contains your original data plus labels for the groups you identified. You can open this file in Excel or other spreadsheet tools.

## 📈 Tips for better results

Data quality matters. If your results look messy, check these points:

* Remove rows that have missing values.
* Ensure all your columns contain numbers.
* Standardize your data if some columns have very large numbers and others have very small ones.
* Try different random states if the results look different each time you run the same method.

The software gives you full control. If one method does not yield a clear pattern, switch to another. Different algorithms look for different signals in the noise. Experimentation is a normal part of the process.

## ⚙️ Handling large files

If your data file is very large, the application might take a few seconds to process the math. You will see a loading indicator on the screen. Avoid interacting with the settings while the indicator appears. This prevents the program from getting stuck. For extremely large files, consider removing unnecessary columns before you upload the data to the app.
